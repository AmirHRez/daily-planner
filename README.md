A personal terminal-based daily planner built with Python and Rich. Plan tomorrow's tasks tonight, track your habits, and check work during the day from terminal (GUI soon)

## Requirements

- Python 3.10+

```bash
pip install rich
```

## How to run

```bash
python main.py
```
Or on **windows**, simply double click on `run.bat`

## Default Habits

The following habits are seeded on the first run and tracked every day:

- English Practice
- Gym / Movement
- AI / ML Learning
- Guitar Practice
- University Study

You can add or deactivate habits directly in the database, or extend `database.py` with a management action if you want to do it from the CLI.
